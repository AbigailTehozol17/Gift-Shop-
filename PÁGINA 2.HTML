PROCESO Y LUGAR DE PRODUCCIÓN DE
NUESTROS ARTÍCULOS

¿Dónde realizamos nuestros productos?

Nuestros productos se realizan directamente dentro de mi hogar, el cual cuenta
con todo el material necesario para realizar cualquier detalle o regalo que
desee nuestro cliente, así como contamos con un área especial, limpia y segura
para realizarlos y que no haya accidentes que provoquen que la imagen del producto
no sea la que esperaba el cliente desde un inicio.

¿Cómo elaboramos nuestros productos?

Para elaborar cada uno de nuestros productos seguimos los siguientes pasos para asegurar su calidad y poder realizar el frasco de dulces selecionado y diseñado por nuestros(as) clientes, los cuales consisten en:
  • Materiales utilizados
    -Frasco
    -Lápiz
    -Lapicero
    -Fomi, papel china, listón, hoja o cartulina donde se escribirá una nota (el color será seleccionado por el cliente)
    -Perforadora de papel.
    -Barras de silicón
    Pistola de silicón
    -Tijeras
    -Dulces
  • 1. El primer paso es elegir el modelo del frasco que haya elegido nuestro(a) cliente para poder lavarlo y limpirarlo correctamente asegurando la higiene de nuestros productos.
  • 2. Posteriormente se comienza a trabajar con los detalles que requerira cada producto, como en ejemplo que es utilizado en las siguientes imagenes, en el cual el objetivo de este regalo es desear un feliz cumpleaños a quien lo reciba. Por lo que con fomi con diamantina se realizan las letras que llevaran el mensaje; el cual junto con el material con el que se elaborará a sido previamente seleccionado por nuestro cliente.
  • 3. Una vez lista la plantilla con la letras que construirán el mensaje que llevara el frasco se procede a recortar lo mas uniforme posible cada una de las letras para que estas tengan un buen aspecto.
  • 4. Después de esto se colocaran las letras recortadas anteriormente sobre el frasco con ayuda de silicón caliente (esto para evitar que se desprendan fácilmente) para formar el mensaje deseado por el/la cliente
  • 5. Dejamos que seque completamente el silicón, y mientras sucede comenzamos a cortar el papel de china para hacer pequeños papeles que decoren la parte inferior del frasco.
  • 6. Luego de esto cortamos la hoja o cartulina del tamaño correcto para que la nota que ha escrito nuestro(a) cliente pueda ser transcrita dentro de este, asegurandonos de dejar un centimetro aproximadamente en la parte superior para del lado izquierdo para poder perforarla sin romperla.
  • 7. Cortaremos el liston de tal forma que pueda darle un par de vueltas al frasco por la parte superior y a su vez sujete de manera correcta sin que dañe el papel donde estará la nota destinada a la persona que recibirá el regalo.
  • 8. Como últimos pasos se comenzará a llenar el frasco con dulces hasta llegar a la parte media de este para volver a colocar papel de china para decorar, este paso se repetirá con la parte superior del frasco, el cual tendrá una cantidad considerable de dulces para darle un detalle especial y un buen servicio a nuestras y nuestros clientes.
